Các cấu trúc điều khiển
1. Cấu trúc lựa chọn / rẽ nhánh
    a) if
        *) Dạng 1: if đơn
            if(condition){
                statement  - condition=true;
            }
        *) Dạng 2: if đầy đủ 
            if(condition){
                statement - condition=true;
            }else{
                statement - condition=false;
            }
        *) Dạng 3: if bậc thang
            if(condition-1){
                statement-1;
            }else if(condition-2){
                statement-2;
            }else.....
                ....
            }else{
                statement-n;
            }
        *) Dạng 4: nested if
            if(condition){
                statement -1;
                if(condition-2){....}
                statement - 2;
            }else{
                if(condition-3){....}
                ....
            }
        
    b) switch...case 
        switch(expression){
            case value1:
                statement-1;
                break;
            case value2:
                statement-2;
                break;
            .....

            [default:
                statement-n+1;
                break;
            ]
        }
2. Cấu trúc lặp:
    a) for - 4 dạng cú pháp
        *) Dạng 1:
            for(initial ; condition ; step){
                statement block;
            }
        *) Dạng 2: for khuyết
            for( ; condition ; step){
                statement block;
            }
            for(initial ;  ; step){
                statement block;
            }
            for(initial ; condition ; ){
                statement block;
            }
        *) Dạng 3: for khuyết toàn bộ
            for( ; ; ){
                statement block;
            }
        *) Dạng 4: for không thân
            for( initial, statement; condition ; step, statement){}
    b) while
        while(condition){
            statement block;
        }
    c) do...while 
        do{
            statement block;
        }while(condition);
    d) foreach
        for(let x in object/array){
            statement;
        }
3. lệnh nhảy
    break;
    continue;
