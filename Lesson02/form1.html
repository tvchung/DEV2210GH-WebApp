Form in HTML5

        Biểu mẫu form 
        * Thẻ Form 
            - Các thuộc tính thường dùng:
                + id, name 
                + action: url (địa chỉ, nơi muốn gửi dữ liệu của đến để xử lý)
                + method: GET / POST / PUT / DELETE
        * Các thành phần của form 
            - Tag input: gồm nhiều loại được quy định qua thuộc tính type 
                type: 
                    text (mặc định):
                        name, id, placeholder, value, maxLength,..
                    password
                        name, id, placeholder, value, maxLength
                    date / datetime-local 
                    checkbox:
                        id, name, checked , value, 
                    radio
                        id, name, checked , value, 
                    number 
                        name, id, placeholder, value, maxLength,
                        max , min 
                    range
                        name, id, placeholder, value, maxLength,
                        max , min 
                    email:
                        name, id, placeholder, value, maxLength,
                        pattern
                    search
                    tel 
                    color 
                    ....

                    submit / reset / button
                
            - Tag Select: name, id,
                    option 
            - Tag textarea
